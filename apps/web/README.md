# ContractorOS Phase 3A Local MVP App Shell

Local React + Vite app shell for ContractorOS California.

## Install

```bash
npm install
```

## Run locally

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Included

- Fixture-only Law & Business practice flow.
- C10 locked/deferred message.
- Source panel.
- Local in-memory issue report mock.
- Blocked route screens.
- Disabled admin placeholder.
- Phase 2E claim governance dashboard using local fixture data.

## Not included

- No backend.
- No Firebase.
- No Airtable API.
- No database.
- No auth.
- No payments.
- No scoring.
- No readiness logic.
- No pass/fail logic.
- No new MCQs.
- No Question Bank migration.
- No deployment.

## Fixture data source

Practice fixture items are the four internal Phase 1V items only. Claim-governance fixtures are derived from the Phase 2E claim eligibility matrix. All data is local and internal-only.
